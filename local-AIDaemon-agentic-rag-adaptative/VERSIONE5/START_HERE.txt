QUÉ ES VERSIONE5
═════════════════

Una implementación mínima de un sistema RAG (Retrieval Augmented Generation)
que combina:

1. BÚSQUEDA VECTORIAL (FAISS)
   └─ Encuentra fragmentos relevantes de documentos

2. EMBEDDINGS (sentence-transformers)
   └─ Convierte texto en vectores numéricos

3. MODELO LLM (llama-cli)
   └─ Genera respuestas basadas en contexto

FLUJO SIMPLE:
   pregunta → buscar → generar respuesta


CARACTERÍSTICAS

LÍMITES (Intencionales)
════════════════════════

✗ Sin chat memory (cada pregunta es independiente)
✗ Sin caché (aún)
✗ Sin evaluadores de calidad
✗ Sin web UI
✗ Sin API REST (aún)

Estos son trade-offs para mantener VERSIONE5 MÍNIMO.
